# HYFIN 리크루팅 관리 툴

한양대학교 금융 동아리 HYFIN의 리크루팅 자동화 툴.
Next.js 15 + Prisma + PostgreSQL(Supabase) + Nodemailer + Vercel.

---

## 서브에이전트 사용 원칙

이 프로젝트에서 코딩 작업은 반드시 서브에이전트에게 위임한다.
에이전트 파일 위치: `.claude/agents/`

### `/implementor` — 코딩 에이전트
코드 작성, 버그 수정, 기능 추가, 리팩터링 담당.
호출 시 반드시 아래 내용을 함께 전달:
- 무엇을 구현/수정해야 하는지 (작업 목표)
- 수정할 파일 경로
- 변경 전/후 코드 또는 변경 방향
- 의존하는 다른 파일이나 API가 있다면 그 내용도 포함

### `/inspector` — 검사 에이전트
버그 탐지, 로직 불일치 검사, 보안 검토 담당. 코드 수정은 하지 않음.
호출 시 반드시 아래 내용을 함께 전달:
- 검사할 파일 목록 또는 범위
- 의심되는 문제 또는 검사 초점
- 의도된 동작 (정답 기준)
