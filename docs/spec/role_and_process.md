기본적으로 admin 페이지는 각자 할당된 id와 password를 통해서만 접근이 가능하도록 구현할거야. 

일단 사용자는 지원자, 운영진, 관리자로 나뉘어.
지원자는 말그대로 apply를 통해 자기소개서를 제출하는 사람이야.
운영진은 admin 페이지에 접근 가능하고, 메일 발송, 면접 관리 시트 수정 등의 권한은 없지만 서류평가와 면접 평가는 가능한 인원들이야. 그리고 면접 관리 시트에서 자신이 평가한 내용만 볼 수 있어.      
관리자는 admin 페이지에 접근 가능하고 운영진보다 더 많은 권한을 가져. 메일 발송 권한, 면접 시트 수정 권한 및 면접 관리 시트의  모든 운영진들의 평가 내용을 볼 수 있는 인원들이야. 이 인원들은 학회장, 부학회장, 대외협력부장, 집행부장, 학술부장 이렇게 5명만 이 권한을 가질 수 있어.
