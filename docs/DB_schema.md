# DB_schema

일단 DB 구조는 아래와 같이 잡아봤는데 너가 생각했을 때 어떻게 구조를 짜면 좋을지 제안해봐.

## 지원자 정보 시트
컬럼은 제출일자 및 제출된 지원서를 통해서 받는 정보들, 자동 접수 확인 메일 발송여부, 서류합불여부, 운영진 통합 서류평가 평균점수, 면접 가능 시간대, 최종합불여부, 운영진 통합 면접평가 평균점수(이 외에 추가적인 컬럼이 필요하다 싶으면 나에게 건의)

## 면접 관리 시트
면접 시간, 공통질문, 최종 개인질문, 운영진별 개인질문, 운영진별 서류평가 점수, 운영진 통합 서류평가 평균 점수, 운영진별 면접점수, 운영진 통합 면접평가 평균 점수, 특이사항(이 외에 추가적인 컬럼이 필요하다 싶으면 나에게 건의)
